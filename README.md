# Voice AI Agent Demo

Basic demo of a voice AI agent for handling inbound service calls.

## Features
- Answers inbound calls automatically
- Collects customer information
- Detects urgency and service type
- Generates structured call summaries

## Use Case
Designed for service businesses such as:
- Plumbers
- Locksmiths
- HVAC professionals

## Goal
Demonstrate how an AI voice agent can improve call handling and reduce missed opportunities.
